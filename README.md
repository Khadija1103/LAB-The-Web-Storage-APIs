# 🧑‍💻 LAB - Web Storage APIs (Pokémon App)

##  Descripción
Este proyecto es una aplicación web que permite buscar Pokémon usando la PokéAPI y guardarlos como favoritos en el navegador usando localStorage.



## Funcionalidades
- Buscar Pokémon por nombre
- Mostrar imagen y datos del Pokémon
- Guardar Pokémon como favoritos
- Evitar duplicados en favoritos
- Guardar datos en localStorage
- Mostrar favoritos al recargar la página


##  Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Bootstrap
- PokéAPI
- Web Storage API (localStorage)


##  API utilizada
https://pokeapi.co/api/v2/pokemon/

---

##  Cómo usar el proyecto
1. Escribe el nombre de un Pokémon
2. Haz clic en "Buscar"
3. Guarda como favorito
4. Revisa la lista de favoritos
